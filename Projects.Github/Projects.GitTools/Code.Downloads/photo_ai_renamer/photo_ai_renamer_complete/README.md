# Photo AI Renamer Tool

This tool lets you:
- Select a folder of images
- Automatically rename them using timestamp and placeholder AI labeling
- Output to a timestamped subfolder
- Log every step

## To Run

1. Install requirements:
```
pip install -r requirements.txt
```

2. Run the app:
```
python gui_main.py
```
