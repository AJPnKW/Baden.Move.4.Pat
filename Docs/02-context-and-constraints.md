# Context And Constraints

## Project Context

This project covered more than a move. It also included:

- staging work for photos and sale viewings
- renovation assessment and option comparison
- fixture and furnishing selection work
- moving in waves during an overlap period
- sequencing around time-sensitive possession dates
- service cutovers and household readiness tasks

## Core Constraints

- fixed dates around possession and move timing
- some work had to happen before full move-in
- renovation choices affected room readiness and sequencing
- some items had to be sold, donated, discarded, or moved by different methods
- not all requirements were known at the start

## Management Approach

The project used a PM-light approach:

- exploratory planning when scope was still forming
- targeted decision logs when tradeoffs mattered
- phased coordination when activities became time-bound
- temporary cutover-style control for tightly coupled changes
