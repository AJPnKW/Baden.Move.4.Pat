# Context And Constraints

## Project Context

This project covered more than a move. It also included:

- staging work for photos and sale viewings
- renovation assessment and option comparison
- fixture and furnishing selection work
- moving in waves during an overlap period
- sequencing around time-sensitive possession dates
- service cutovers and household readiness tasks
- household transition decisions that affected storage, disposal, sale, donation, and move scope
- medical-support and supply continuity that required coordination rather than last-minute improvisation

## Core Constraints

- fixed dates around possession and move timing
- some work had to happen before full move-in
- renovation choices affected room readiness and sequencing
- some items had to be sold, donated, discarded, or moved by different methods
- not all requirements were known at the start
- budget and timing tradeoffs had to be balanced together, not separately
- some decisions had real deadline pressure because waiting too long removed viable options

## Management Approach

The project used a PM-light approach:

- exploratory planning when scope was still forming
- targeted decision logs when tradeoffs mattered
- phased coordination when activities became time-bound
- temporary cutover-style control for tightly coupled changes

## Delivery Modes Used

Two delivery modes were used within the same overall effort.

### Agile / Exploratory Mode

Used when:

- options were still being investigated
- scope was still being clarified
- decisions needed proof-of-concept or validation first
- partial progress still created value

Examples:

- renovation option analysis
- fixture and furnishing research
- early move-lane thinking
- deciding what should be sold, donated, stored, or moved

### Cutover / Big-Change Mode

Used when:

- partial completion would break the whole sequence
- timing windows were fixed
- occupancy readiness depended on multiple moving parts landing together

Examples:

- possession day and immediate stabilization
- utility and connectivity cutovers
- final mover-day preparation
- pre-move readiness tied to renovation dependencies
