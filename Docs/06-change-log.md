## Change-Control Note

This project used lightweight change control rather than formal change boards.

The practical test was:

- does this change alter only one local area, or
- does it affect timing, dependencies, room readiness, move sequencing, or service cutover?

Local changes were handled with adjustment.
Cross-cutting changes were treated as higher-control changes.
